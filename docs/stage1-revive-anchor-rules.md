# Stage 1 锚点复活规则说明

## 固定锚点

- `B1_SafeZone`
- `1F_Lobby`
- `7F_Bench`

## 复活流程

1. 记录死亡（地点/死因/掉落）
2. 选择重开或复活
3. 解析最近已解锁锚点（图距离）
4. 同步快进 12h（时间、任务/NPC/服务刷新）
5. 执行掉落转移（deterministic）
6. 写回快照并异步推进 world engine

## 掉落归属规则（确定性）

- 优先归击杀者
- 否则归当前房间主威胁
- 否则归最近 hostile
- 否则进入 lost pool

## 验收要点

- 复活后玩家位置与锚点一致
- 时间确实 +12h
- 玩家行囊扣除，掉落有归属记录
- 异步推进不阻塞玩家返场

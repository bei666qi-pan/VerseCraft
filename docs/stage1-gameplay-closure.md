# Stage 1 玩法闭环说明

## 闭环目标

- 新开局可在 B1 获得自然任务引导
- 上楼可感知强度与收益爬升
- 死亡可重开或复活，且复活具备代价与世界推进
- NPC 关系、图鉴、任务引用能进入系统状态

## 玩家主链路

1. 开局进入 B1 安全中枢  
2. 通过服务节点/NPC 获取基础任务  
3. 上楼探索（任务、奖励、风险提升）  
4. 死亡后选择重开或复活  
5. 复活触发 12h 快进 + 物品转移 + 世界状态推进  
6. 继续推进主线/角色线/阴谋线

## 第一阶段边界

- 不新增物理房间节点
- 不引入重型新 infra（如强依赖新向量库）
- 不追求完整经济系统与职业系统终版

# 文界工坊（VerseCraft）

> **不是你在读故事，是故事在试图读懂你。**

一款以 **规则怪谈** 为骨、以 **属性检定** 为血、由 **AI 地下城主** 实时推进的单机文字冒险游戏。

你会坠入 **如月公寓**。<br>
七层楼，七层异响，七层越来越不讲理的规则。<br>
你能做的不是背攻略，而是输入一句话，赌它会不会成为你活下去的关键。

**一句话安利：** 像在玩一场会临场接招、会反将你一军、还会越玩越压迫的怪谈跑团。

## 它为什么容易让人上头

- **一句话就能推进剧情**：不用写长段 RP，像说人话一样输入动作就能继续。
- **每次选择都真有代价**：说错一句、慢一步、赌错一次，都可能直接出事。
- **规则不是摆设**：这里的守则不是背景板，而是会真的咬人的东西。
- **属性不是数字装饰**：敏捷、幸运、魅力、出身，会决定你能不能骗过去、逃出去、撑到最后。
- **AI 不只是陪聊**：它会接住你的奇怪操作，也会放大你的错误判断。
- **退出不是暂停**：这里没有“安全离场”，只有活着通关，或者死着结算。

## 30 秒理解玩法

1. 先铸造你的角色，分配属性，选择回响天赋。
2. 进入如月公寓，阅读当前局势和可选行动。
3. 输入一句自然语言动作，比如：

> 我先别出声，贴着墙往前挪。<br>
> 我敲门三下，然后退后半步。<br>
> 我把手电照向走廊尽头，不进门，先听。

4. AI 地下城主会根据你的动作、属性、时间推进和世界规则，立刻给出后果。
5. 活下去，摸清真相，或者在某个你没想到的瞬间被规则清算。

## 你会在这里体验到什么

- **压迫感**：楼里不是一直在吓你，而是一直在等你自己出错。
- **参与感**：你不是翻页看剧情的人，而是亲手把剧情推向未知的人。
- **不确定性**：同样一句“我开门看看”，不同角色、不同时间点，后果都可能完全不同。
- **悬念感**：你永远不知道下一条守则是在救你，还是在骗你。

## 适合谁

- 喜欢规则怪谈、惊悚网文、文字跑团的人。
- 喜欢“我的选择会改变结果”，而不是纯阅读剧情的人。
- 想体验 AI 当 DM 时那种临场反馈、随机性和压迫节奏的人。
- 想找一个一句话就能开玩的中文单机沉浸式冒险的人。

## 快速开始

```bash
pnpm install
pnpm dev
```

先把 `.env.example` 复制为 `.env.local` 并按模板填好配置。默认会在 `http://localhost:666` 启动，进入「铸造角色」后就可以直接开局。

### 项目定位

VerseCraft 是一个单机、浏览器内运行的中文文字冒险项目。它把规则怪谈叙事、属性检定和 AI 驱动的临场回应揉在一起，目标不是做“技术演示”，而是做“会让人一直想再试一把”的体验。

### 技术栈

- Next.js 16 + React 19
- Tailwind CSS v4
- Zustand 5
- PostgreSQL + Drizzle
- IndexedDB（`idb-keyval`）做客户端持久化

### 本地开发常用命令

```bash
pnpm dev
pnpm build
pnpm test:unit
pnpm test:e2e:chat
pnpm run ship -- "feat: your message"
```
